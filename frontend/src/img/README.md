# Изображения (`src/img`)

Подключаются через `src/utils/localImages.js`.

| Файл | Назначение |
|------|------------|
| `апполон для welcomepage на фон.png` | Welcome: подложка **слева** (под контентом) |
| `фото чуть выше центра на главной.JPG` | Welcome: **под бейджем**, в потоке, без прозрачности |
| `картинка вопрос для заполнения пустого места страниц снизу.png` | Низ онбординга (слева в паре) |
| `воскл знак для заполнения пустого места страниц снизу.png` | Низ онбординга (справа от «вопроса») |
| `апполон на фон страниц тренировки.png` | Полоса **над** «Упражнения» на Home + фон PlanGenerating |
| `меню снизу …` | Нижнее меню Home: **гантель** → заметки → main → питание → настройки |

Расширения: `png`, `jpg`, `jpeg`, `webp`, `svg`, `JPG`, `PNG`.
