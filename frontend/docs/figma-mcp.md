# Figma → вёрстка (Vue / Ionic)

## Статус MCP

В Cursor может быть подключён сервер **`figma-developer-mcp`** (часто как `user-http://127.0.0.1:3333/sse`).

Инструмент **`get_figma_data`** ходит в Figma REST API: `GET /v1/files/:file_key`.

### Почему ссылка вида `/site/...` не подходит

URL **Figma Sites**:

`https://www.figma.com/site/<id>/...`

— это другой продукт. Для него **тот же `file_key` часто даёт `400 Bad Request`** при вызове API файлов дизайна.

**Что сделать:** открыть макет как **Figma Design** (файл `.fig`) и скопировать ссылку вида:

`https://www.figma.com/design/<FILE_KEY>/Название?node-id=123-456`

- **`FILE_KEY`** — сегмент в URL после `/design/`
- **`node-id`** в URL: `123-456` → в MCP/API обычно **`123:456`**

После этого `get_figma_data` сможет забрать дерево слоёв и текст/размеры для вёрстки.

## Ключ API

**Не передавайте ключ в чат и не держите его в истории терминала.** Лучше:

```bash
set FIGMA_API_KEY=...
npx figma-developer-mcp --figma-api-key=%FIGMA_API_KEY%
```

или переменная окружения, которую подхватывает ваш MCP-конфиг в Cursor.

Если ключ уже светился в открытом терминале — имеет смысл **отозвать и выпустить новый** в настройках Figma.

## Если MCP недоступен

Пришлите **скрины экранов** + по возможности экспорт ассетов (SVG/PNG) и значения из Dev Mode (отступы, шрифты, цвета) — вёрстка возможна и без MCP.

## Картинки из Figma (экспорт)

**Да, выгрузить можно** — у ассистента нет прямого доступа к вашему Figma-аккаунту, но есть три рабочих варианта:

1. **REST API Images** (тот же `FIGMA_API_KEY`, что у MCP):
   - `GET https://api.figma.com/v1/images/:file_key?ids=1:2,3:4&format=png` (или `svg`, `jpg`)
   - В ответе — временные URL; их нужно скачать и положить, например, в `frontend/public/img/`.
   - `ids` — node id в формате **`1:2`**, как в API (из `node-id=1-2` в URL).

2. **MCP `figma-developer-mcp`** — если в вашей сборке MCP есть инструмент вроде **`download_figma_images`** / экспорт изображений по node id, им можно скачать файлы в проект при запущенном сервере MCP (см. описание инструментов в Cursor → MCP).

3. **Вручную в Figma:** выделить фрейм / слой → **Export** (PNG/SVG/PDF).

Если пришлёте **file key** и список **node id** нужных иллюстраций, можно добавить в репозиторий скрипт `npm run figma:images`, который дернёт API и сохранит файлы в `public/`.
