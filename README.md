# Fitness App

Мобильное приложение с ИИ-тренером для тренировок в зале.

## Структура проекта

- **backend/** — Go API (GORM + PostgreSQL)
- **ai/** — Python часть (FastAPI + ИИ-генерация планов и чат)
- **frontend/** — Vue 3 + Capacitor (мобильное приложение)
